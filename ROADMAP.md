# AI Markdown Mirror Roadmap

## v0.1.0 MVP

- Dynamic `.md` pages.
- Head discovery links.
- HTTP Link header.
- `/llms.txt`.
- `/llms-full.txt`.
- Settings page.
- Per-content exclude controls.
- Default noindex headers.

## v0.1.1 Stabilization

- Fix bugs found on TroyMaya staging.
- Improve route handling for nested pages/custom permalinks.
- Improve Markdown conversion for WordPress blocks.
- Add clearer admin notices.

## v0.2.0 Better Admin UX

- Drag-and-drop post type ordering.
- Indexed counts per post type.
- Better settings layout.
- Preview buttons for LLM and Markdown URLs.

## v0.3.0 Advanced Content Controls

- Per-post custom LLM title/description.
- Category/tag include/exclude controls.
- Custom post type templates.
- Optional sitemap integration.

## v0.4.0 Podcast / Transcript Mode

- Preserve chapter lists.
- Preserve transcript accordions.
- Generate optional podcast-focused LLM file.

## v0.5.0 Pro Candidate Features

- Advanced cleanup selectors.
- Multilingual/hreflang support.
- Gravity Forms field extraction.
- AI readiness scanner.
- Scheduled regeneration/cache warming.
- Agency import/export settings.
